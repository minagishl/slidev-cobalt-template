# Slidev Cobalt Template

A [Slidev](https://github.com/slidevjs/slidev) starter template using [slidev-theme-cobalt](https://github.com/minagishl/slidev-theme-cobalt) — a professional dark presentation theme for developers.

## Getting Started

```bash
bun install
bun run dev
```

Then visit <http://localhost:3030>

Edit [slides.md](./slides.md) to customize your presentation.

## Available Scripts

| Script           | Description                  |
| ---------------- | ---------------------------- |
| `bun run dev`    | Start the development server |
| `bun run build`  | Build for production         |
| `bun run export` | Export slides to PDF         |
